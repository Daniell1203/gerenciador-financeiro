#  Gerenciador Financeiro

Projeto simples feito em Python para controlar saldo pelo terminal.

## Funcionalidades

- Adicionar salário
- Adicionar gastos
- Ver saldo atual
- Categorias de gastos:
  - Comida
  - Transporte
  - Lazer

## Tecnologias

- Python 3

## Objetivo

Projeto criado para praticar:
- `while`
- `if/elif`
- lógica de programação
- entrada de dados
- menus no terminal
